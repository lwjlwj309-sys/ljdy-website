# 律金盾官网

债事综合服务平台官方网站

## 技术栈

- **框架**: Astro
- **样式**: Tailwind CSS
- **部署**: Vercel（推荐）/ Nginx

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署到 Vercel

1. 将代码推送到 GitHub
2. 在 [vercel.com](https://vercel.com) 导入项目
3. 自动部署完成
